## 背景・課題・解決されること
関西といえばたこ焼き</br>
全国にもっとタコパを広げたい</br>
たこ焼きを盛り上げれば関西も盛り上げると考えてこのアプリを作りました</br>
このアプリではたこ焼きの写真を撮り、その写真から画像認識を用いて点数化し、ランキングとして表示できます</br>
ランキング化することによってたこ焼き競争心が芽生え、関西をええ感じにできます！</br>
